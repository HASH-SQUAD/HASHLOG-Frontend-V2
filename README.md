# HASHLOG
## 프로젝트 설명 ✅
- 전공동아리 #ASH의 블로그입니다. 팀에서 있었던 일들을 정리하여 올리고, 팀원 자신의 아티클을 공유하기 위해 기획하게 되었습니다.

## 개발기간 👶🏻
> 24.04.23 ~ ing

## 기능설명 💬

### main
- 게시된 글들을 볼 수 있습니다. <br>
- 글쓰기 버튼을 통해 글을 올릴 수 있습니다. <br>
- 로그인을 할 수 있습니다.<br>
<img width="1000" alt="image" src="https://github.com/HASH-SQUAD/HASHLOG-Frontend/assets/128370710/f71b4ae8-63d8-41d8-bd1d-a17d0cc64c3f">


### signin
- 아이디, 비밀번호를 입력하여 로그인을 할 수 있습니다.<br>
<img width="1000" alt="image" src="https://github.com/HASH-SQUAD/HASHLOG-Frontend/assets/128370710/f2eeb813-c538-4561-ac36-a967d6735e62">


### signup
- 회원이 아닐 경우, 회원가입을 진행 할 수 있습니다. <br>
<img width="1000" alt="image" src="https://github.com/HASH-SQUAD/HASHLOG-Frontend/assets/128370710/c2bf0dfa-5246-421e-8501-678f5b3803b4">


### write
- 마크다운으로 글을 작성 할 수 있습니다.
- 왼쪽에서 글을 작성하면 오른쪽 화면에 미리보기로 보여지게 됩니다.
- 현재 태그 부분 개발 진행중입니다 !
<img width="1000" alt="image" src="https://github.com/HASH-SQUAD/HASHLOG-Frontend/assets/128370710/de821135-cc1b-470c-b4b7-03fedc51623a">

### post
- 작성된 글을 확인 할 수 있습니다.
- 공유하기 버튼을 누르면 모달이 뜨며 url이 복사 됩니다.
